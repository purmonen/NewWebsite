Armada

Monafide is currently developing Armadas iOS-app. Armada is a career fair where students at KTH meet companies. The app displays information about the companies and information about Armada. Focus is enhancing the user experience by following iOS Human Interface Guidelines and established design principles.

Xlnc Hockey

During the fall of 2014 Monafide developed the iOS-app for XLNC Hockey. The user can perform different excercises such as skating and stick handling. The app collects sensor data from the accelerometer in order to analyse the users performance. The app was developed following a strict specification on functionality and design.

mySKILAB

During the summer of 2014 Monafide developed mySKILABs iOS-app. Using mySKILAB the user can record skiing trips and get feedback on technique and performance. The app collects data from sensors and analyse the data using a motion analysis engine in the cloud.

Allsvenskan

One developer is making an app for following swedish football in his spare time. Features include live score, game schedules, score tables and forums. The backend scrapes the web, provides a JSON api and sends push notifications when new data is available.

Raspberry Pi

During the one day Ericsson E-Hack 2015 Monafide developed an iOS-app for displaying temperature and camera images collected from a cluster of Raspberry Pis.
