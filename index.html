Djenee

Monafide is currently developing Djenees iOS-app. Djenee is a digital concierge service that does everything you want on demand. The user wishes something and then Djenee will return with an offer. If the user accepts the offer the user can pay to make the wish come true. The app is available on iPhone and Apple Watch.

Armada

Armada is a careers fair where companies can come to KTH in order to market themselves to students. Monafide is currently developing Armadas iOS-app that will help students find suitable companies and work as a guide for students attending the fair. The focus when developing the app is on enhancing the user experience by following iOS Human Interface Guidelines and established design principles.

Xlnc Hockey

During the fall of 2014 Monafide developed the iOS-app for XLNC Hockey. The app is intended to help ice hockey players optimise their training by allowing the user to perform different exercises such as skating and stickhandling. It collects sensor data from the iPhone's built-in accelerometer and gyroscope in order to analyse the user's performance and was developed following a strict specification on functionality and design. The app was completed on time and within budget.

mySKILAB

After developing an early version in Objective-C as students at KTH, Monafide was contracted to continue developing mySKILAB for WEMEMOVE in Swift during the summer of 2014. Using mySKILAB the user can record skiing trips and get feedback on technique and performance. The app collects data from bluetooth and built-in sensors in the iPhone and analyses the data using a motion analysis engine in the cloud. The app was completed on time and within budget.

Allsvenskan

One of our developers is making an app for following swedish football in his spare time. Features include live score, game schedules, score tables and forums. The backend scrapes the web, provides a JSON api and sends push notifications when new data is available. The app is available on iPhone and Apple Watch.

Raspberry Pi

During the one day Ericsson E-Hack 2015 Monafide developed an iOS-app for displaying temperature and camera images collected from a cluster of Raspberry Pis.
